Data Source

Please take into consideration that with any data refresh it will count towards your API Requests.

API requests are as follows:

  • When you are first loading the page, it generates one request to pull the list of coins dynamically
  • When clicking on a tracker image, it will generate 3 requests to pull 3 types of history charts
  • When it refreshes the data to pull the information for trackers, it will generate only one request for all tracked coins
  • When you check for your rates, you will do one request
  • Auto refresh pulls only once, every X amount of time. It's just like a manual refresh, meaning you only multiply by the amount of time it runs.

You can calculate and see your API calls in realtime on Crypto Compare or use the Data Source Rate Limits option from the Crypto Dash settings.

Commands
  • set-data-source {VALUE} would set the data source
  • set-data-apikey {value} would set the data api key (trim white space from value)
  • show-data-apikey would bring the current api key if set
  • show-data-rate would bring up the current data rates
  • show-data-source would bring up the available data sources

  • add-tracker {TRACKERCOIN}
  • remove-tracker {TRACKERCOIN}
  • set-tracker-size {value} would set the width of the tracker
  • show-tracker-details {TRACKERCOIN} would bring up the tracker info even if the tracker info is not loaded in the array.

  • set-currency {VALUE} would change currency to USD/GBP/EUR (command for currency setup only allow current currencies)

  • set-datarefresh-auto would turn on data to automatically refresh
  • set-datarefresh-manual would set data to manual refresh
  • set-datarefresh-auto {value} would turn on the data to auto refresh with the specified value

  • set-frosted-mode {value} would turn on or off the frost (true/false)
  • set-color-mode {value} would turn on or off the dark mode (light/dark)
  • set-color-theme {value} would set the color theme
  • set-classic-background {value} would set the background to classic and selected value. Turn on autocomplete for current value
  • set-animated-background {value} would set the background to animated and selected value. Turn on autocomplete for current value

  • set-notification-orientation {value} would set the orientation (top/bottom)
  • set-notification-location {value} would set the orientation (left/right/center)
  • set-notification-sound {value} would turn on or off the alarm sound (true/false)

  • cryptodash-reset-config would reset the config
  • cryptodash-save-config would save the config
  • cryptodash-load-config would bring up the the load config dialog
